# ЕГЭ Обществознание Бот

Бот для подготовки к ЕГЭ по обществознанию с функциями решения задач (1 и 2 части), добавления своих задач, рейтингом и интеграцией с нейросетью DeepSeek для проверки развёрнутых ответов.

## Функционал

- Решение задач 1 части (автоматическая проверка по эталону)
- Решение задач 2 части (проверка через DeepSeek API с выставлением баллов и комментарием)
- Добавление задач (для себя или на модерацию для всех)
- Чёрный список задач (исключить надоевшие задачи)
- Рейтинг задач (пользователи оценивают задачи, формируется средний балл)
- Система монетизации (опционально, отключается в конфиге)

## Установка и запуск

1. Клонируй репозиторий

2. Установи зависимости

        pip install -r requirements.txt

3. Настрой конфиг  
   Скопируй `config.example.yaml` в `config.yaml` и заполни:
   - `telegram.token` – токен от BotFather
   - `deepseek.api_key` – ключ DeepSeek (если используешь проверку 2 части)
   - `monetization.enabled` – включить/отключить проверку подписки

4. Создай базы данных

        python data/initialize.py

5. Запусти бота

        python src/main.py

## Работа с задачами

- Основные задачи хранятся в `tasks.db` (общие и личные).
- Добавленные пользователями задачи для всех попадают в `query.db` (модерация).
- Для проверки задач из очереди модерации используй `data/tasks_checker.py` (см. ниже).

## Модерация задач

Скрипт `data/tasks_checker.py` помогает админу просматривать задачи, добавленные пользователями, и решать: добавить в общую базу или удалить.

### Запуск

        python data/tasks_checker.py

### Работа

- Скрипт подключается к `query.db` и берёт первую непроверенную задачу.
- Выводит все поля: номер (num), условие, ответ/критерии, ID пользователя.
- Админ вводит:
  - `accept` – перенести задачу в `tasks.db` (user_id = NULL, рейтинг 0)
  - `reject` – удалить задачу из очереди
  - `exit` – завершить проверку
- После каждого действия скрипт переходит к следующей задаче (если есть).

## Примечания

- Для работы DeepSeek API требуется интернет и валидный ключ (бесплатные токены доступны при регистрации).
- Если монетизация выключена, все функции доступны всем пользователям.
- Чёрный список работает по ID задач (не по номерам).
- Личные задачи видны только добавившему их пользователю.
